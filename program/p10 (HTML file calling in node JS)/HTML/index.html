HTML Forms






Forms Element

1. The "input" Element

* <input type="text"> :- is used to create a single-line text input field where users can enter textual data.

Attributes

A. id :- The HTML id attribute is used to specify a unique id for an HTML element. You cannot have more than one element with the same id in an HTML document.



========================================================

B. name :- name Attribute is used to specify a name for an <input> element. It is used to reference the form-data after submitting the form or to reference the element in a JavaScript.






========================================================

C. value :-Specifies the default value of the input field.




========================================================

D. placeholder :- Provides a hint to the user of what can be entered in the input field.




========================================================

E. maxlength :- Defines the maximum number of characters allowed in the input field. Textarea element. Its default value is 524288.




========================================================

F. required :- Indicates that the input field must be filled out before submitting the form.




========================================================

G. readonly :- Makes the input field read-only, preventing the user from modifying its value.







========================================================

H. disabled :- Disables the input field, making it uneditable and excluding it from form submission.
